3D Human Motion Authoring and Editing


    A simplistic comprehensive application system to effortlessly author, edit, and validate human motion data. The system enables a naive user to interactively edit existing motion data using a humanoid model in a 3D space based on user-defined scenarios and synthesize numerous variations of motion sequences.
